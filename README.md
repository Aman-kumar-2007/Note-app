# 📝 Notes App with Todo List

A simple and efficient Notes Application that allows users to create, manage, and organize notes along with a built-in Todo List feature. This project demonstrates basic CRUD operations and task management using JavaScript and Local Storage.

---

## 🚀 Features

### 📌 Notes (CRUD)

* ➕ Create new notes
* 📖 View existing notes
* ✏️ Edit/update notes
* ❌ Delete notes

### ✅ Todo List

* ➕ Add tasks
* ✔️ Mark tasks as completed
* 🗑️ Delete tasks
* 📋 Manage daily activities

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Local Storage (for data persistence)

## 💡 How It Works

* Notes and todos are stored in browser Local Storage
* Data remains saved even after refreshing the page
* Users can perform full CRUD operations on notes
* Todo items can be marked complete or deleted
